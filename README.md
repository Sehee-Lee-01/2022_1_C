## 2022.03.16

- ==과 =은 다르다.
- 줄바꿈 안해도 괜찮습니당.  “;”표시로 구분함.

- 선언문과 실행문 구분
```c++
int a[10];// 선언문
a[10] = 0 // 실행문, logical error
```
- 선언시 생략 가능한 것
```c++
a[n][m] = {...}
/*n: 생략가능
m: 생략 불가능(중괄호 안에 유추 불가해서 )
	*/
```
